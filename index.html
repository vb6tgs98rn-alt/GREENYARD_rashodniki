Списание

  
База знаний
Полное руководство — читай при первом использовании
Green Yard
Приложение для учёта расходников по нескольким квартирам посуточной аренды. Видишь сколько чего осталось, когда пополнять и автоматически списываешь при каждом заезде.
Квартиры
Каждая квартира — отдельный объект со своими расходниками и историей.
Расходники
Два типа: Одноразовые и Не одноразовые. Для первых настраивается расход на заезд, для вторых — количество в комплекте.
Быстрые действия
Левая панель → раздел «Быстрые действия».
История
Все действия фиксируются автоматически. Открыть: ☰ → «История списаний».
Заявки на закупку
Список того что нужно купить. Открыть: ☰ → «Заявки на закупку».
Сохранение данных
Приложение работает офлайн, данные хранятся прямо в браузере.
